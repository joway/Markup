# Block

基于 PLY , 改良 Markdown 语法 .
 
---

## 需求

### 排版需求
1. 水平居中 , 垂直居中
2. 居左，居右
3. 图片

### 定制需求
1. 添加绑定插件
2. 添加函数


## 语法

    {{ funciton: param1, param2 }}
    
    {% func funciton %}
    param string
    {% endfunc funciton %}
    
    {% func funciton: param2, param3 %}
    param string
    {% endfunc funciton %}
